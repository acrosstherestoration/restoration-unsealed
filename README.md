# Restoration Unsealed

![Restoration Unsealed Banner](./public/first_vision.png)

A Digital Research Hub for RLDS History and Restoration Branch Theology.

**Live Site:** [https://acrosstherestoration.github.io/restoration-unsealed/](https://acrosstherestoration.github.io/restoration-unsealed/)

## Overview

Restoration Unsealed is a specialized archive and research interface designed to provide modern scholars and seekers with direct access to curated primary and secondary sources from the Restoration movement. 

The hub leverages **Google NotebookLM** to provide an intelligent, source-grounded research experience across several key thematic archives.

## Curated Research Notebooks

- **📜 The Historical Record (1830–1890):** Primary source archives, multi-volume histories, and early periodicals.
- **🕊️ The Living Faith:** A century of sermons, testimonies, and modern faith practice.
- **📖 Restoration Scriptures & Doctrine:** The Three Standard Books and official doctrinal expositions.
- **🛡️ The Case for the Reorganization:** Legal history, authority, and rebuttals of historical departures.
- **🖋️ Restoration Scriptures - Hebrew and Greek Scholar:** Advanced linguistic analysis using lexicons, linguistic analysis, and cultural commentaries.

## Features

- **Direct Notebook Integration:** Seamlessly transition to advanced AI-powered research environments.
- **Thematic Navigation:** Easily find sources related to history, faith, or advanced scriptural analysis.
- **Polished Interface:** A design inspired by vintage academic archives and modern digital aesthetics.

## Local Development

If you wish to run the research hub interface locally:

**Prerequisites:** Node.js

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Run the app:**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.
